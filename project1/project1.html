Notes for Lesson 1

1.World Wide Web

The World Wide Web is mainly composed of HTML documents. HTML represents Hypertext Markup Language, which is used by the browers to parse the webpage. The major pieces of the World Wide Web include clients, Internet and servers. And a client interacts with servers with HTTP.

2.HTML Elements

HTML Elements are used to indicate how the browers will show the contents in the webpage.

Most of HTML Elements are composed of openning tag, closing tag, and contents between them.

<b>text</b> makes the contents Bold
<em>text</em> makes the contents Italic
ps: how to show < and > are learned from http://www.w3schools.com/HTML/html_entities.asp

3.HTML Attribute

Some elements contain attributes in the opening tag, which is used to provide extra information for the browser. For example, the anchor tag uses a href attribute to define the URL to link.

<a href="url">link</a> establishes a hyperlink, whose attribute href is url and name is link.
<img src="url" alt="text"> shows an image whose address is url. If the image cannot be shown, then "text" will be shown.

4.More HTML Elements

<p>text</p> defines a paragraph. The contents are contained in a box.
<br> is a void element and is used to switch to a new line.

<div>content</div> is a block element.
<span>content</span> is an inline element.

1.Tree Structure

The definition in the following paragraph is from wikipedia article on Tree Structures

A tree structure or tree diagram is a way of representing the hierarchical nature of a structure in a graphical form. It is named a "tree structure" because the classic representation resembles a tree, even though the chart is generally upside down compared to an actual tree, with the "root" at the top and the "leaves" at the bottom.
